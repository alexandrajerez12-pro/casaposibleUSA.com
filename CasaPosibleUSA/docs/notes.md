# Casa Posible USA Project Notes

## Overview
This document contains notes and documentation related to the Casa Posible USA project. It serves as a reference for developers and contributors.

## Project Structure
- **index.html**: Main entry point for the website.
- **css/styles.css**: Styles for the main project.
- **js/script.js**: JavaScript for interactivity.
- **archive/**: Contains subprojects.
  - **subproject-1/**: First subproject with its own HTML, CSS, and JS files.
  - **subproject-2/**: Placeholder for the second subproject.
- **assets/**: Resources such as fonts and SVGs.
- **docs/**: Documentation files, including this notes file.
- **.gitignore**: Specifies files to ignore in version control.
- **LICENSE**: Licensing information for the project.
- **README.md**: Overview and usage instructions.
