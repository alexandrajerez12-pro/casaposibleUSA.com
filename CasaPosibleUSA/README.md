# CasaPosibleUSA

Sitio estático del proyecto "Casa Posible USA".

## Contenido

- `index.html`: Archivo HTML principal que sirve como punto de entrada para el sitio web.
- `css/styles.css`: Archivo que contiene los estilos para el proyecto principal, definiendo la presentación visual de los elementos HTML.
- `js/script.js`: Archivo que contiene el código JavaScript para el proyecto principal, manejando la interactividad y el contenido dinámico.
- Carpeta `archive/`:
  - `subproject-1/`: Primer subproyecto dentro del archivo.
    - `index.html`: Archivo HTML principal para el primer subproyecto.
    - `css/styles.css`: Estilos específicos para el primer subproyecto.
    - `js/script.js`: Código JavaScript para el primer subproyecto.
  - `subproject-2/`: Carpeta designada para el segundo subproyecto (sin archivos listados).
- Carpeta `assets/`:
  - `fonts/`: Carpeta destinada a almacenar archivos de fuentes utilizados en el proyecto.
  - `svgs/`: Carpeta destinada a almacenar archivos SVG utilizados en el proyecto.
- Carpeta `docs/`:
  - `notes.md`: Archivo que contiene documentación o notas relacionadas con el proyecto.
- `.gitignore`: Archivo que especifica archivos y directorios que deben ser ignorados por Git.
- `LICENSE`: Archivo que contiene la información de licencia para el proyecto.

## Cómo usar localmente

1. Abrir `index.html` en un navegador (sitio estático).
2. O servir con un servidor estático, por ejemplo:

```
python3 -m http.server 8000
```
